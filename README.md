# OpenCar

OpenCar是一个文件处理工具，主要用于对视频和图片文件进行字节反转处理，支持文件预览和下载功能。

## 功能特点

- 📁 支持上传视频和图片文件
- 🔄 对文件进行字节反转处理
- 🎥 实时预览处理后的文件
- 💾 下载处理后的文件
- 📱 响应式设计，支持移动端和高分辨率设备

## 技术栈

- Vue 3
- Element Plus
- JavaScript
- CSS3

## 使用方法

1. 打开应用
2. 点击上传区域或拖拽文件到上传区域，选择要处理的视频或图片文件
3. 点击"转换并下载"按钮，将处理后的文件下载到本地
4. 点击"预览（图片/视频）"按钮，实时预览处理后的文件

## 注意事项

- 本站不提供任何文件资源，所有操作均在本地完成
- 处理大文件可能需要较长时间，请耐心等待
- 支持的文件类型：视频（mp4、webm等）和图片（jpg、png、gif等）
